# LaimiuTrade 需求文档

个人波段交易复盘日志系统。胜利目标：完成 50 个 30% 波段的复利增长。

## 一、目标系统（核心）

- **单位净值法记账**：账户视作一只基金，初始净值 1.0。入金按当前净值折算申购份额，出金按当前净值赎回份额，入出金不影响收益率曲线。
- **50 个胜利节点**：净值达到 1.3^1 ~ 1.3^50 依次点亮。
  - 节点状态**实时判定**：当前净值跌破 1.3^n，第 n 个节点熄灭。进度反映"此刻真实战果"。
  - 界面同时展示每个节点按当前份额换算的资金金额。
- **节点事件史**：每次点亮/熄灭都记录时间戳。统计页展示"首次达成耗时"趋势图（第 n-1 → n 节点耗时），耗时缩短代表进步。
- 首页常驻：当前净值、已点亮节点数、距下一节点差距。

## 二、交易记录

- 结构化流水：日期、标的代码、名称、方向（买/卖）、价格、数量。
- 费用自动计算，费率可配置：佣金费率（含最低佣金）、卖出印花税、过户费。
- **回合归组**：同一标的从建仓（持仓 0 → 正）到清仓（回到 0）自动归组为一个回合，自动计算回合盈亏。胜率、盈亏比、连胜连亏按回合统计。
- **截图导入**：上传同花顺持仓/成交截图 → 视觉大模型识别 → 进入"待确认"列表 → 人工核对后入库。保留手动录入。

## 三、四层复盘体系（固定模板）

| 层级 | 栏目 |
|---|---|
| 每日复盘 | 当日交易（自动带出）、收盘总资产快照（手动填）、盘面观察、决策复盘、错误教训、K线截图粘贴、操作打分（6 维度） |
| 次日预研 | 大盘预判、关注标的+触发条件、仓位计划、风险预案 |
| 周复盘 | 本周收益/回撤（自动）、交易汇总（自动）、做对的事、做错的事、下周策略 |
| 月复盘 | 月度数据+节点进度（自动）、体系迭代、下月目标 |

- 漏写复盘的交易日提醒/标红。
- 每日/周/月复盘可导出排版好的 PDF 分享。

## 四、AI 能力（可优雅降级）

- **AI 操作打分**：结合当日行情、交易记录、复盘文字，按 6 维度打分并点评：
  仓位控制、回撤控制、计划执行力、买点质量、卖点质量、情绪管理。
  AI 分数与点评仅作参考，最终分可手动修改。
- **截图识别**：视觉大模型识别同花顺截图（见二）。
- 模型服务：OpenAI 兼容接口，可配置 Base URL + API Key + 模型名（文本模型、视觉模型各一个配置位）。
- 未配置 Key / 断网：核心记账复盘功能照常，AI 功能停用，打分可手动填写。

## 五、行情数据（三源可配置）

优先级链路可配置，默认：本地通达信 → akshare → 网络抓取（东方财富公开接口）。

- 本地通达信 `vipdoc` 目录直读日线（路径可配置，默认 `D:\new_tdx\vipdoc`）。
- 用途：AI 打分的行情上下文、大盘/个股当日数据展示。

## 六、灵感闪记

- 随时记录的轻量卡片，带标签分类。
- 首页每天随机展示一张旧卡片温故。

## 七、图表统计与数据

- 净值曲线 + 回撤曲线（标注节点线）。
- 日/周/月收益率、最大回撤、回合胜率、盈亏比、连胜连亏、节点耗时趋势图。
- SQLite 本地存储；一键导出 JSON/Markdown 备份。

## 八、软件形态与技术

- 本地网页应用，单人使用，免登录。
- 后端 FastAPI + SQLite，前端 React (Vite)。
- 双击脚本启动，浏览器访问。

## 实施顺序

1. 核心记账 / 净值 / 节点系统
2. 复盘体系
3. 图表统计
4. AI 能力（打分、截图识别）
5. 导入 / 导出（PDF、备份）
